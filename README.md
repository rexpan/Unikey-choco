# Unikey

[Chocolatey](chocolatey.org) [package](https://chocolatey.org/packages/Unikey) for [Unikey](http://unikey.org).

# Install

```
choco install unikey
```

# Build

```
choco pack
```

# Debug

```
choco install Unikey -fdvy -s '%cd%'
```

# Publish

```
choco push -s https://push.chocolatey.org
```
